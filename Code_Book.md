##Description
This is a document that specifies the variables, identifiers and data used during the project. 

##Dataset
The description of the data source can be accesed on the following [link] (http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones)
The dataset consist in the measurements of a Samsung Galaxy's sensors in various trial subjects.

##Identifiers
* **SUBJECT** the person who is using the phone during the observations
* **ACTIVITY** the activity the subject is doing while using the phone.
  * **WALKING** (value 1): subject was walking during the test
  * **WALKING_UPSTAIRS** (value 2): subject was walking up a staircase during the test
  * **WALKING_DOWNSTAIRS** (value 3): subject was walking down a staircase during the test
  * **SITTING** (value 4): subject was sitting during the test
  * **STANDING** (value 5): subject was standing during the test
  * **LAYING** (value 6): subject was laying down during the test
